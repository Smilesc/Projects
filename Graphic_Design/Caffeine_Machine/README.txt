To run:
1. Click the dropdown menu next to "Beverage:", select an option
2. Click the dropdown menu next to "Volume:", select an option
3. When you have made both selections, press "Go"
4. If desired, click the dropdown menu next to "Cup Caffeine Limit:" and choose
    an amount. This represents the height of the cup andwill set what your beverage 
    caffeine amounts are scaled to. Press "Set"
